### Моделирование случайных событий (GUI)

**Часть 1:**  
Приложение «Скажи “да” или “нет”».

**Часть 2:**  
Приложение «Шар предсказаний» (Magic 8-Ball).



### Реализация

Для моделирования используются функции модуля `random`:

1. Модуль «Да / Нет» (Взвешенный выбор)
- Используется функция `random.choices(population, weights, k=1)`.
- Пространство исходов: Ω={ДА,НЕТ}
- Вероятность зафиксирована программно: P(ДА)=0.7, P(НЕТ)=0.3.

2. Модуль «Шар предсказаний» (Полнотекстовый динамический выбор):

В данной версии реализована возможность тонкой настройки вероятности для каждого из 12 возможных исходов через отдельный графический интерфейс (окно настроек).

- Интерфейс: Для сохранения чистоты основного окна настройки весов вынесены в дочернее окно (Toplevel), где пользователь с помощью регуляторов (Scale) определяет «вес» каждого ответа.
- Метод: Взвешенная выборка из полного списка фраз.
- Пространство исходов: Ω={A1​,A2​,…,A12​}.
