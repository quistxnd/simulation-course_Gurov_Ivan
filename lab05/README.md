### Моделирование случайных событий (GUI)

**Часть 1:**  
Приложение «Скажи “да” или “нет”».

**Часть 2:**  
Приложение «Шар предсказаний» (Magic 8-Ball).



### Реализация

Для моделирования используются функции модуля `random`:

1. Модуль «Да / Нет» (Взвешенный выбор)
- Используется функция `random.choices(population, weights, k=1)`.
- Пространство исходов: Ω={ДА,НЕТ}

2. Модуль «Шар предсказаний» (Равномерный выбор):
- Используется функция `random.choice(sequence)`.
-  Пространство исходов: Ω={A1,...,A12}
