### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализация

Для генерации последовательности псевдослучайных чисел используется линейный конгруэнтный метод, описываемый рекуррентным соотношением:

$$x_{n+1} = (a \cdot x_n + c) \pmod{m}$$
$$\alpha_n = \frac{x_n}{m}$$

Выбранные параметры:
- Модуль ($m$): $2^{32}$ (определяет максимально возможный период).
- Множитель ($a$): $1\,664\,525$.
- Приращение ($c$): $1\,013\,904\,223$.
- Начальное значение (seed): $42$.

### Теоретические показатели

Базовый датчик должен имитировать непрерывное равномерное распределение на отрезке $[0, 1]$. Для такого распределения теоретические моменты вычисляются следующим образом:

Математическое ожидание ($M$):
$$M[X] = \frac{a + b}{2} = \frac{0 + 1}{2} = 0.5$$
Дисперсия ($D$):
$$D[X] = \frac{(b - a)^2}{12} = \frac{1^2}{12} \approx 0.083333$$
