### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализация

Был использован линейный конгруэтный генератор (LCG), а также для сравнения был взят генератор из модуля `random`, который был основан на алгоритме MT19937.

### Результаты

| | Среднее | Дисперсия|
|----------------------|--|-|
| LCG | 0.501241 | 0.083046  |
|MT19937 | 0.498888 | 0.082908 |
|Теоретические результаты| 0,5|0,083333|

### Вывод

- Были реализованы базовый датчик и встроенный генератор
- Вычисленные выборочные среднее и дисперсия для обоих генераторов близки к теоретическим значениям для равномерного распределения на
[0,1].
- Полученные результаты подтверждают корректность реализации и соответствие законам математической статистики при большом объёме выборки (100 000 значений).
