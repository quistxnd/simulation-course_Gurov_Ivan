### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализация: 64-битный линейный конгруэнтный генератор

В данной работе реализован линейный конгруэнтный генератор. Выбраны 64-битные параметры, предложенные Д. Кнутом:

$$X_{n+1} = (a \cdot X_n + c) \mod m$$
$$\alpha_n = \frac{X_n}{m}$$

### Параметры генератора:

| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| **m** | $2^{64}$ | Модуль (определяет максимально возможный период) |
| **a** | $6\,364\,136\,223\,846\,793\,005$ | Множитель (параметр Кнута) |
| **c** | $1\,442\,695\,040\,888\,963\,407$ | Приращение |
| **seed** | Текущее системное время | Начальное состояние |


### Теоретические показатели

Для непрерывного равномерного распределения на интервале $[0, 1]$ характеристики имеют фиксированные значения:

1. **Мат. ожидание:**
   $$M[X] = \frac{a + b}{2} = \frac{0 + 1}{2} = 0.5$$

2. **Дисперсия (Разброс):**
   $$D[X] = \frac{(b - a)^2}{12} = \frac{1}{12} \approx 0.083333$$


### Результаты тестирования при n = 100000

В ходе выполнения программы были получены следующие статистические данные:

### Анализ свойств реализованного датчика

#### 1. Соответствие распределению
Выборочные мат. ожидание (0.49992) и дисперсия (0.08309) реализованного алгоритма практически идентичны теоретическим значениям. Отклонение мат. ожидания составляет менее 1%, что подтверждает высокую точность моделирования равномерного распределения.

#### 2. Отсутствие корреляции
Проверка автокорреляции между соседними членами выборки показала значение примерно равное 0.000898. Близость этого значения к нулю подтверждает статистическую независимость последовательных чисел (отсутствие линейной зависимости между ними).

#### 3. Апериодичность
Использование 64-битного модуля $2^{64}$ гарантирует большой период повторения последовательности. Тестовая выборка в $10^5$ элементов составляет крайне маленькую долю периода, что исключает возможность зацикливания или повторов.

#### 4. Воспроизводимость и скорость
- Алгоритм обеспечивает полную воспроизводимость результатов при фиксации начального числа (seed).
- Генерация всей выборки (100000 чисел) произошла за за 0.035 сек, что подтверждает высокую эффективность метода.


### Вывод

Реализованный линейный конгруэнтный генератор полностью соответствует требованиям, предъявляемым к базовым датчикам случайных чисел. Статистические показатели практически совпали с теоретическими ожиданиями и результатами встроенного генератора Python.
