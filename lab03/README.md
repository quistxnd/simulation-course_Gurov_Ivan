### Клеточные автоматы. Лесные пожары (GUI)

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.

### Описание

Данная программа представляет собой модель распространения лесного пожара, реализованную с использованием двумерного клеточного автомата на языке Python с графическим интерфейсом (Tkinter). Поле моделирования состоит из клеток, каждая из которых может находиться в одном из состояний: пустая клетка, дерево, огонь или пепел. Состояние клеток изменяется по заданным правилам на каждом шаге симуляции.

В модели учитываются дополнительные факторы, влияющие на распространение огня: направление ветра, уровень влажности, рост новых деревьев и случайные возгорания (молнии). Также реализовано движущееся дождевое облако, которое может тушить пожар. Пользователь может управлять параметрами симуляции через графический интерфейс и наблюдать изменение состояния системы в реальном времени.
